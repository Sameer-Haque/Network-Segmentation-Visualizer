# Network-Segmentation-Visualizer

Web based network segmentation visualizer that monitors network traffic through SPAN/mirror ports or a sniffer and scanning, analyzes segmentation and traffic in real time (?), while providing insights on things like usage alerts, configuration issues,  etc... 

Rough flow:
Setup endpoint
Setup sending
Setup receiving
Setup analysis
Draw fancy pictures.
